ğŸ”§ TROUBLESHOOTING
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

PROBLEM: "ModuleNotFoundError: No module named 'PySide6'"
CAUSE: PySide6 not installed
SOLUTION:
1. Open terminal in project folder
2. Run: pip install PySide6
3. Run: python main.py

PROBLEM: "Python is not recognized as an internal or external command"
CAUSE: Python not in system PATH
SOLUTION (Windows):
1. Reinstall Python
2. During installation, CHECK "Add Python to PATH"
3. Restart computer
4. Try again

SOLUTION (Mac/Linux):
1. Use python3 instead of python
2. Try: python3 main.py

PROBLEM: "Permission denied" or "Access is denied"
CAUSE: No write permissions
SOLUTION (Windows):
1. Right-click Command Prompt
2. Select "Run as administrator"
3. Navigate to folder and try again

SOLUTION (Mac/Linux):
1. Use: sudo python main.py
2. Enter your password

PROBLEM: Application starts but seems slow
CAUSE: Computer resources
SOLUTION:
1. Close other applications
2. Restart computer
3. Clear temporary files

PROBLEM: "No such file or directory"
CAUSE: Wrong folder path
SOLUTION:
1. Make sure you're in the saki-doruma folder
2. Check: ls (Mac/Linux) or dir (Windows)
3. You should see: main.py, config.py, data/, modules/, ui/
